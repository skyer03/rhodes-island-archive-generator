# 罗德岛人事档案生成器

一个可直接发布的静态网页小工具，用来填写并导出竖版 PNG 人事档案图。

## 使用方式

直接用浏览器打开 `index.html` 即可使用。发布到 GitHub Pages、Netlify、Vercel 或任意静态网站托管时，把本目录中的文件一起上传即可。

## 首版功能

- 本地填写档案资料并实时预览。
- 上传干员头像和名片 / 展示照片。
- 身高、体重留空时自动显示分离式黑色遮盖小方块。
- 导出 `1080 x 1600` 的 PNG 图片。
- 不登录、不联网、不上传，所有资料只在当前浏览器中处理。

## 素材说明

页面使用的罗德岛标志来自用户提供的本地素材，位于 `assets/rhodes-logo.png`。用户上传的头像和展示图仅在本地浏览器中使用。
